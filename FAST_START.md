# RigForge（锐格锻造坊）Fast Start

## 1. 准备

在仓库根目录执行后续命令。

## 2. 启动服务（前台）

```bash
./scripts/server.sh restart fg
```

启动后访问：

- [http://127.0.0.1:8000](http://127.0.0.1:8000)

## 3. 常用命令

```bash
./scripts/server.sh restart bg   # 后台启动/重启
./scripts/server.sh stop         # 停止服务
./scripts/server.sh status       # 查看服务状态
```
