# 商业分析（中国装机方向）

**这个项目存在的意义不大。至少当下没看到出路。**

**第一，中国装机用户的需求相对简单，不需要复杂的功能。**
**第二，国内装机用户的主要需求是办公和游戏，而不是专业的游戏或专业的工作。**
**第三，装机用户要么追求降低成本，要么追求极致性能配置；低预算的以办公为主，预算相对较低，且容易被商家以低价诱惑，这压缩了商业空间，这类群体没有服务价值。高预算的也未必自己设计复杂的装机方案，RIG党多是跟谁某项配置。**
**第四，国内装机用户的主要问题是购买时的决策困难，因为他们对技术细节不了解，也没有专业的推荐。品牌机和品牌组装机已经红海血战了。**
**第五，国内商业环境不易复刻PCPartPicker+NewEgg商业模式。**
